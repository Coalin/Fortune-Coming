# 量化选股系统 - 特征文档

## 1. 概述

本系统使用 **约218个特征**，用于预测股票未来14日的收益涨跌。

### 特征分类

| 类别 | 数量 | 说明 |
|------|------|------|
| 价格技术指标 | 20+ | 均线、RSI、MACD、布林带、ATR等 |
| 短期收益序列 | 60 | Ret_1day ~ Ret_30day（对数收益率） |
| 短期成交量序列 | 30 | VolChg_1day ~ VolChg_30day（对数变化率） |
| 动量特征 | 80+ | 多周期动量、斜率、加速度、一致性等 |
| 时间序列编码 | 9 | 连涨/连跌、波动率比、熵、Hurst指数等 |
| 机构量化因子 | 31 | 来自顶级量化基金实践 + 国泰君安191因子库 |

## 2. 核心技术指标

### 2.1 移动平均线 (MA)

| 特征 | 说明 |
|------|------|
| MA5, MA10, MA20, MA30 | 简单移动平均 |
| MA5_Slope ~ MA120_Slope | 各周期均线斜率 |
| MA5_Trend ~ MA120_Trend | 各周期均线趋势方向 |
| MA5_Week_Slope ~ MA20_Week_Slope | 周线均线斜率 |

**计算公式**：
```python
MA_n = 收盘价的n日简单平均
MA_Slope = (MA_n - MA_n_shift_n) / (MA_n_shift_n * n)
```

## 6. 标签定义

```python
label = 1  # 正样本：未来14日收益 > 10% 且 回撤 > -10%
label = 0  # 负样本：未来14日收益 <= -3%
排除: -3% < 收益 <= 10% 的样本
```

## 7. 特征选择流程

```
原始特征 (~218个)
    ↓
① 相关性筛选 (阈值=0.9)
    ↓ 剔除高相关特征对
剩余特征
    ↓
② 重要性筛选 (Top 50)
    ↓ 按重要性排序取前50
最终训练特征 (50个)
```

## 9. 优缺点分析

### 优点

1. **特征全面**：覆盖价、量、动量、趋势、波动率等多维度
2. **机构级别因子**：引入顶级量化基金实践因子
3. **科学筛选**：相关性 + 重要性双重筛选
4. **时间序列适配**：包含Hurst指数、趋势持续性等时序特征

### 缺点

1. **纯价量数据**：缺乏基本面因子（PE、PB、ROE等）
2. **缺乏另类数据**：无舆情、资金流、卫星等数据
3. **单市场**：未考虑跨市场联动
4. **未做行业中性化**：特征可能受行业因素影响

### 改进方向

1. 加入基本面因子（需财务数据API）
2. 加入北向资金、龙虎榜等资金流数据
3. 加入舆情/新闻情绪因子（需NLP接口）
4. 考虑行业市值中性化处理
