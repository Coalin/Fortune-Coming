# 量化选股系统 - 特征文档

## 1. 概述

本系统使用 **173个特征**，用于预测股票未来14日的收益涨跌。特征主要分为以下几大类：

| 类别 | 数量 | 说明 |
|------|------|------|
| 价格技术指标 | 12 | 均线、RSI、MACD、布林带等 |
| 市场结构特征 | 17 | 行业相对强弱、资金流向等 |
| 短期收益序列 | 60 | 过去1-30日收益率、成交量变化 |
| 动量特征 | 84 | 多周期动量、动量加速度、动量一致性、交易信号、均线趋势等 |

---

## 2. 价格技术指标

### 2.1 移动平均线 (MA)

| 特征 | 说明 |
|------|------|
| MA5 | 5日简单移动平均 |
| MA10 | 10日简单移动平均 |
| MA20 | 20日简单移动平均 |
| MA30 | 30日简单移动平均 |

**用途**：判断短期趋势方向，MA向上交叉看涨，向下交叉看跌。

### 2.2 相对强弱指数 (RSI)

| 特征 | 说明 |
|------|------|
| RSI | 14日RSI（默认） |

**计算公式**：
```
RSI = 100 - (100 / (1 + RS))
RS = 平均涨幅 / 平均跌幅
```

**用途**：
- RSI > 70：超买，可能下跌
- RSI < 30：超卖，可能上涨

### 2.3 MACD指标

| 特征 | 说明 |
|------|------|
| MACD | 快EMA - 慢EMA（12,26） |
| MACD_Signal | MACD的9日EMA |

**用途**：MACD金叉买入，死叉卖出。

### 2.4 布林带 (Bollinger Bands)

| 特征 | 说明 |
|------|------|
| BOLL_upper | 上轨 = MA20 + 2×标准差 |
| BOLL_mid | 中轨 = MA20 |
| BOLL_lower | 下轨 = MA20 - 2×标准差 |

**用途**：价格触及下轨可能反弹，触及上轨可能回调。

### 2.5 其他技术指标

| 特征 | 说明 | 计算方式 |
|------|------|---------|
| ATR | 平均真实波幅 | TR的14日均值 |
| OBV | 能量潮 | 累积成交量加权和 |
| CCI | 商品通道指数 | (TP-SMA)/MAD×0.015 |
| VOLATILITY | 波动率 | 20日收益标准差 |

## 7. 特征选择策略

系统使用 **Top 35 特征选择**：

1. 先用全部144个特征训练临时模型
2. 获取特征重要性（weight）
3. 选择重要性排名前35的特征
4. 用精选特征进行正式训练

这样可以：
- 减少过拟合风险
- 降低训练时间
- 提高模型泛化能力

## 8. 特征工程注意事项

### 8.1 缺失值处理
- 使用中位数填充
- 无限值替换为NaN后再填充

### 8.2 异常值处理
- 数值限制在 [-1e10, 1e10] 范围内

### 8.3 特征标准化
- XGBoost对特征尺度不敏感，无需标准化
- 逻辑回归需要标准化（本系统未使用）

## 9. 特征与标签的关系

### 9.1 标签定义
```python
label = 1  # 正样本：未来14日收益 > 10% 且非高波动
label = 0  # 负样本：未来14日收益 <= -3%
```

### 9.2 正负样本比例
- 正样本：收益 > 10% 且非高波动
- 负样本：收益 <= -3%
- 排除区间：-3% ~ 10%（56%左右被排除）

### 9.3 特征与预测目标的关系

| 特征类型 | 预期与标签关系 |
|---------|--------------|
| 正动量特征 | 高动量 → 可能正收益 |
| 负动量特征 | 低动量/反转 → 可能正收益 |
| 波动率特征 | 高波动 → 排除（高风险） |
| 资金流特征 | 正向资金流 → 支撑上涨 |
